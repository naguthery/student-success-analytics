# student-success-analytics

A data analytics project that examines university student performance to identify factors associated with academic success, course outcomes, instructor effectiveness, and program performance. The project combines SQL, Python, Tableau, and data modeling to transform raw data into actionable insights for higher education stakeholders.

## Project Overview

Student success is influenced by many academic and institutional factors. This project analyzes a relational university dataset to evaluate student performance, instructor effectiveness, course outcomes, and program success.

The project demonstrates an end-to-end analytics workflow that includes:

* Data modeling with an Entity Relationship Diagram (ERD)
* Data preparation using Python
* SQL analysis using DuckDB in Google Colab
* Interactive dashboard development in Tableau
* Business recommendations based on analytical findings

---

## Business Problem

Universities rely on data to improve student outcomes and allocate resources effectively. This project investigates several key questions:

* Which programs have the highest pass rate?
* What factors are most associated with student success?
* Which instructors have the highest pass rate?
* Which courses have the highest failure rates?
* Which students are at risk of failing?
* What are the retention rates by student cohort?
