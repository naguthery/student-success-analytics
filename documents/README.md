# Student Success Reports

This folder contains the reports and Entity Relationship Diagram (ERD) created for the Student Success Analytics project. These documents describe the project's data model, analysis methodology, and key findings related to student, instructor, course, and program performance.

## Contents

- 'Student Success Analytics Full Report.pdf'– A comprehensive report detailing the project objectives, methodology, data analysis, dashboard visualizations, key findings, and recommendations.

- 'Student Success Analytics Case Study.pdf' – A condensed version of the full report that summarizes the business problem, analytical approach, key insights, and recommendations.

- 'Student Success Analytics ERD.pdf' – An Entity Relationship Diagram (ERD) illustrating the database schema and the relationships between the project tables.
