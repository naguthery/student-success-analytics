# Student Success Data

This folder contains the datasets used for the Student Success Analytics project. The data was cleaned, transformed, and analyzed to support the evaluation of student, instructor, course, and program performance.

## Contents

- `Course_Sections.csv` – Contains information about individual course sections.
- `Courses.csv` – Contains information about the courses offered by the university.
- `Departments.csv` – Contains information about the university's academic departments.
- `Enrollments.csv` – Contains student enrollment records for each course section.
- `Grades.csv` – Contains the grades earned by students in their enrolled courses.
- `Instructors.csv` – Contains information about instructors who teach university courses.
- `Programs.csv` – Contains information about the academic programs offered by the university.
- `Student_Support.csv` – Contains information about student support services.
- `Students.csv` – Contains demographic and academic information about students.
- `Terms.csv` – Contains information about the academic terms in which courses were offered.
